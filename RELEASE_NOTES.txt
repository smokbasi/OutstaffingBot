OutstaffingBot — release notes (staging)

2026-06-23 — Phase 9: модерация контента и админ
- Модерация текста заявок и профилей: wordlists (profanity, sex, drugs, translit), обфускация, translit, alcohol-whitelist
- Нарушения логируются; после N попыток (MODERATION_VIOLATION_THRESHOLD) — очередь модерации
- Админ-команды: /moderation_queue, /violation_log, /block_user, /unblock_user
- Админ-команды: /admin_stats, /unverified_employers, /verify_employer, /audit_log
- Новые работодатели verified=false — опубликовать заявку только после /verify_employer
- audit_log: block/unblock, employer.verify

2026-06-22 — Mini App: работодатель принимает отклики
- В режиме «Я работодатель» появилась вкладка «Отклики» — список откликов на ваши заявки
- Кнопки «Принять» и «Отклонить» для статуса «на рассмотрении»
- API: GET /employer/applications, PATCH accept/reject; при принятии — синхронизация постов в группах

- «📝 Заполнить профиль»: если анкета уже есть — показываем профиль и кнопку «✏️ Обновить профиль», а не сброс FSM с нуля
- Ссылка t.me/...?start=job_{id}: зарегистрированным сразу карточка вакансии, без повторной анкеты
- Единая проверка «профиль заполнен» (имя, метро, ставка, опыт или флаг resume_completed)

2026-06-22 — Отклик из групповых постов через бота
- Кнопка «Откликнуться 👷» в группе ведёт в бота: t.me/Outstaffing_Work_BOT?start=job_{id}
- Бот проверяет профиль: без анкеты — FSM регистрации, затем карточка вакансии
- Отклик на смену — inline-кнопки; заявка появляется в «Мои отклики» (бот + Mini App)

2026-06-22 — Исправлена публикация вакансий в Telegram-группы
- В группах WebApp-кнопки не поддерживаются — URL-кнопка t.me/...?start=job_{id}
- Корректная сборка deep link URL при MINI_APP_URL с query (?v=2)
- /register_group корректно сохраняет группу при апгрейде в supergroup (TelegramMigrateToChat)
- Ошибки отправки в группу пишутся в лог worker/bot

2026-06-22 — Mini App: исправлены кнопки выбора роли
- Экран «Кто вы?»: заголовок и подзаголовок снова на отдельных строках, нормальная высота карточек
- Восстановлены стили списков, чипов и карточек заявок (сломанный CSS после настроек уведомлений)

2026-06-22 — Регистрация групп только в чате группы
- /register_group, /unregister_group, /group_status — только в группе (не в личке)
- После /register_group бот пишет в группу: «✅ Группа привязана» с категориями
- В личке: /admin_groups — список зарегистрированных групп

2026-06-22 — Phase 6: умное закрытие/открытие постов в группах
- Если работник снимает отклик — пост в группе снова открывается (кнопка «Откликнуться», без «Закрыто»), если вакансия ещё активна и людей не набрано
- Когда принято нужное число людей — пост автоматически закрывается, как при ручной отмене заявки
- Работодатель может принять отклик через API (PATCH /employer/applications/{id})

2026-06-22 — Phase 6: публикация в Telegram-группы
- Админ-команды: /register_group, /unregister_group, /group_status, /admin_groups
- Автопост вакансии в зарегистрированные группы при публикации заявки (если «в группы: да»)
- Кнопка «Откликнуться» открывает Mini App
- При закрытии/отмене заявки пост в группе обновляется — «Закрыто»
